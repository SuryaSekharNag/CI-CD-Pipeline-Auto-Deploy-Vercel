# CI-CD-Pipeline-Auto-Deploy-Vercel 🚀

This project demonstrates a complete CI/CD pipeline using:

- Next.js
- Jenkins
- GitHub Webhooks
- Vercel Deployment

## 📌 Features

- Automated build using Jenkins
- GitHub integration with webhook triggers
- Continuous deployment to Vercel
- Modern Next.js application
